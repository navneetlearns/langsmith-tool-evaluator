Eval Testing Principles (HEART.md)

  1. Think like a user — send the query, capture what comes back. Errors are real results.
  2. Wait patiently — SSE timeout 300s. Multi-turn, data-driven responses take time.
  3. Measure response time — every query records response_time_seconds.
  4. Versioned reruns — query_results_v{N}.jsonl. Never overwrites. Compare across versions.
  5. No retry, no masking — if a query fails, that IS the result.
  6. Full trace capture — tool calls, SSE status sequence, response, suggestions, errors, timing.
